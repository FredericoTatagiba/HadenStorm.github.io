  • "Call" the variables

So, to start a program, first you need to "call" your variables, that means you will put if it will be a whole number (Example: 1), a real number (Example: 1.5) or if it will be a word.

  1. If you want that to be a whole number, you can use int, for example. But remember, you can't put broken number (like 1.5 or 1.0 too).
    2. 					
#include <stdio.h>
##include <stdio.h>

int main()
{
	int n1=1, n2;

// Using %i in your printf, you can print n1. 
	printf("%i", n1);
//And using scanf, you can make the person, who is using your program, to choose a number, for example:
	printf("Choose a whole number for n2: "); scanf("%i", &n2);
//And also print it too. 
	printf("%i", n2);
	return 0;
}
  2. Or, if you a real number, you use float or double. And, diferent from int, float/double can receive whole numbers too.
    3. 					
#include <stdio.h>
##include <stdio.h>
					
int main()
{
	float n1=1, n2;

// Using %f in your printf, you can print n1. 
	printf("%f", n1);
//And again, using scanf, you can make the person, who is using your program, to choose a number.
	printf("Choose a real or a whole number for n2: "); scanf("%f", &n2);
//And print it too.
	printf("%f", n2);
/*And you can also choose how many bnumbers you want after the dot, using %.nf 
where "n" is the number of number that you want to be shown.*/
	printf("%.2f", n2);
	return 0;
}

Now that you know how to call a variable'and how to print and scan it, let's learn some fuctions, like conditions to print something and even to make the process repeat.

  1. Condition

    2. To put it simple, it's a fuction that will let you print something if the conditions were met. For exemple:

    					
#include <stdio.h>
##include <stdio.h>

int main()
{
	int x, y;

	printf("\n\n\tChoose a whole number for x: "); scanf("%i", &x);
	printf("\n\n\tChoose a whole number for y: "); scanf("%i", &y);
//First, you use "if" to put the condition between the parentheses
	if(x < y)
		printf("\n\n\tX is smaller than Y");
//And then you can use "else" to do an action if the coditions weren't met
	else
	{
//You can also put another codition, but it needs to be "inside" in an "else"
		if (y < x)
			printf("\n\n\tX is smaller than Y");
//And you put another else inside
		else
			printf("\n\n\tX = Y");
	}
	return 0;
}
  2. Repetition

    3. Repetition is a loop, that will continue if a condition is met. An example of it would be a "Yes/No" questions like "do you want this message to be shown again?"

    					
#include <stdio.h>
##include <stdio.h>

int main()
{
	int x=1;
	char option;

//If the variable doesn't have a value yet, you use "do while".
	do
	{
		printf("\n\n\tDo you want to continue (Y/N)? "); scanf(" %c", &option);
	}while(option=='Y' || option=='y');

//if it already has a value, you can use only while if you want
	while(x==1 || x!=0)
	{
		printf("\n\n\tShow this message again?\n\t1. Yes\n\t0. No\n\tOption: "); scanf("%i", &x);
		if(x!=1 && x!=0)
			printf("\n\n\tInvalid option.");
  	}
    return 0;
}
